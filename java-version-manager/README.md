## Java Version Manager

### How to compile from source: 

```bash
$ cd previous/java-version-manager
$ make
$ ./jvmanager <arg>
```

GOAL: 

* Automatically detects OS
* Installs binaries in the native package manager (e.g. apt, dnf, scoop, brew)

OS SUPPORT: 

* Debian & Ubuntu
